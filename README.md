# 🎵 音乐下载器

一个的QQ音乐下载工具，支持单曲和歌单下载，具备智能风控、多线程并发等高级特性。

## 🌟 主要特性

### 🔧 核心功能
- **智能下载**：支持QQ音乐单曲和歌单下载
- **多线程并发**：智能线程池管理，提升下载效率
- **进度显示**：实时显示下载进度和速度 (`xx% xx.x/xx.xMB xx.xKB/s`)
- **智能去重**：自动识别并去除重复下载任务
- **断点续传**：文件已存在时自动跳过，避免重复下载

### 🛡️ 安全防护
- **智能风控**：随机延迟、User-Agent轮换、真实请求头模拟
- **输入验证**：严格验证歌曲ID格式，拒绝歌曲名输入
- **长度限制**：防止恶意超长字符串输入
- **配置检查**：自动验证下载目录和Cookie文件配置
- **安全过滤**：防范SQL注入、路径遍历等恶意输入

### 📊 统计分析
- **精准统计**：准确记录成功/失败次数和成功率
- **错误分类**：智能归类相似错误，避免重复记录
- **详细日志**：按日期生成日志文件，便于问题排查
- **实时反馈**：下载过程中的详细状态信息

### 💻 GUI界面
- **图形化操作**：直观的图形界面，支持单曲和歌单下载
- **实时进度条**：在GUI中显示每个下载任务的实时进度
- **状态反馈**：清晰显示下载状态（已存在、下载中、完成、失败）
- **错误处理**：统一的错误提示弹窗
- **任务结束提示**：下载完成后显示"任务结束"提示
- **配置管理**：可在GUI中配置Cookie文件和下载位置
- **帮助信息**：提供详细的使用帮助

## 💡 GUI使用指南

### 启动界面
1. 启动GUI后会显示"使用须知"弹窗，阅读后点击"我已知晓并同意"继续
2. 进入欢迎界面，点击"Next"按钮开始使用

### 配置Cookie
1. 点击菜单栏"配置" -> "配置cookie"
2. 系统会打开cookie配置文件供您编辑
3. 如果没有cookie文件，会自动创建并打开

### 配置下载位置
1. 点击菜单栏"配置" -> "下载位置"
2. 在弹出的对话框中选择您希望保存音乐的目录
3. 选择有效的目录后，音乐将保存到该位置
4. 如果未配置或配置无效的下载位置，系统会在下载前提示"请参阅帮助 配置有效的下载位置"

### 查看帮助
1. 点击菜单栏"帮助" -> "帮助"查看使用说明
2. 帮助文档包含了如何配置下载位置的相关说明

### 服务选择
1. 选择"下载歌单内所有歌曲"或"下载我需要的歌曲"
2. 输入歌单链接或歌曲ID（多个ID用空格分隔）

### 下载任务
1. 输入相应信息后点击"开始下载"
2. 实时查看下载进度和状态
3. 下载完成后会弹出"任务结束"提示

## ⚠️ 注意事项

### Cookie配置
- Cookie是下载QQ音乐资源的必要凭证
- Cookie过期时需要重新获取
- 请妥善保管Cookie信息，不要泄露给他人

### 下载位置配置
- 请确保配置了有效的下载位置且具有写入权限
- 如果未配置或配置无效，系统将提示配置下载位置
- 可通过"配置"->"下载位置"菜单项进行设置

### 风控提醒
- 频繁下载可能触发平台限流
- 建议合理使用下载功能，避免过度请求
- 本工具仅供个人学习和备份使用

### 网络要求
- 稳定的网络连接
- 足够的存储空间
- 支持HTTPS的网络环境

## 🛠️ 配置说明

### 基本配置 (config/settings.py)
```python
DOWNLOAD_CONFIG = {
    'target_dir': r'C:\Users\YourName\Music',  # 下载目录
    'cookie_path': 'cookies_y.qq.com.txt',     # Cookie文件路径
    'format': 'bestaudio/best',                # 音频格式
    'max_retries': 3,                          # 最大重试次数
    'min_delay': 1,                           # 最小延迟(秒)
    'max_delay': 3,                           # 最大延迟(秒)
    'download_location': r'C:\Users\YourName\Music'  # 下载位置配置
}
```

### Cookie文件获取
1. 登录QQ音乐官网
2. 按F12打开开发者工具
3. 刷新页面，在Network标签页找到cookie信息
4. 将cookie保存到 `cookies_y.qq.com.txt` 文件

## 🔍 错误处理

### 常见错误类型
- **无效ID**：输入格式不正确的歌曲ID
- **网络超时**：网络连接不稳定导致的超时
- **权限不足**：下载目录无写入权限
- **配置缺失**：缺少必要的配置文件

### 错误恢复机制
- 自动重试（最多3次）
- 指数退避延迟策略
- 详细的错误日志记录
- 友好的用户提示信息
