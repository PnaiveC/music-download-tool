# 🎵 音乐下载器

一个的QQ音乐下载工具，支持单曲和歌单下载，具备智能风控、多线程并发等高级特性。

## 🌟 主要特性

### 🔧 核心功能
- **智能下载**：支持QQ音乐单曲和歌单下载
- **多线程并发**：智能线程池管理，提升下载效率
- **进度显示**：实时显示下载进度和速度 (`xx% xx.x/xx.xMB xx.xKB/s`)
- **智能去重**：自动识别并去除重复下载任务
- **断点续传**：文件已存在时自动跳过，避免重复下载

### 🛡️ 安全防护
- **智能风控**：随机延迟、User-Agent轮换、真实请求头模拟
- **输入验证**：严格验证歌曲ID格式，拒绝歌曲名输入
- **长度限制**：防止恶意超长字符串输入
- **配置检查**：自动验证下载目录和Cookie文件配置
- **安全过滤**：防范SQL注入、路径遍历等恶意输入

### 📊 统计分析
- **精准统计**：准确记录成功/失败次数和成功率
- **错误分类**：智能归类相似错误，避免重复记录
- **详细日志**：按日期生成日志文件，便于问题排查
- **实时反馈**：下载过程中的详细状态信息

## 🛠️ 配置说明

### 基本配置 (config/settings.py)
```python
DOWNLOAD_CONFIG = {
    'target_dir': r'C:\Users\YourName\Music',  # 下载目录
    'cookie_path': 'cookies_y.qq.com.txt',     # Cookie文件路径
    'format': 'bestaudio/best',                # 音频格式
    'max_retries': 3,                          # 最大重试次数
    'min_delay': 1,                           # 最小延迟(秒)
    'max_delay': 3,                           # 最大延迟(秒)
}
```

### Cookie文件获取
1. 登录QQ音乐官网
2. 按F12打开开发者工具
3. 刷新页面，在Network标签页找到cookie信息
4. 将cookie保存到 `cookies_y.qq.com.txt` 文件

## 🔍 错误处理

### 常见错误类型
- **无效ID**：输入格式不正确的歌曲ID
- **网络超时**：网络连接不稳定导致的超时
- **权限不足**：下载目录无写入权限
- **配置缺失**：缺少必要的配置文件

### 错误恢复机制
- 自动重试（最多3次）
- 指数退避延迟策略
- 详细的错误日志记录
- 友好的用户提示信息
