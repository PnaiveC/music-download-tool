# GUI模块

## 项目结构

```
ui/
├── gui.py                 # 主GUI界面文件，包含所有UI组件
├── test_gui.py           # GUI测试文件
├── __init__.py           # 模块初始化文件
└── README.md             # 本说明文件
```

## 功能描述

GUI模块是音乐下载工具的图形用户界面部分，使用PyQt6框架开发，提供了现代化、响应式的用户界面，包含以下功能：

1. **启动公告弹窗**
   - 展示使用须知和法律条款
   - 用户需同意后才能使用软件

2. **欢迎页面**
   - 展示欢迎信息
   - 提供进入服务选择的入口

3. **服务选择页面**
   - 提供两种服务选项：歌单下载和单曲下载
   - 包含动画效果和交互反馈

4. **下载任务页面**
   - 根据用户选择提供相应的输入界面
   - 显示下载进度

## 设计特点

- **响应式布局**: 界面能够适应不同窗口大小
- **现代化UI**: 使用了现代设计元素，如圆角按钮、颜色渐变等
- **动画效果**: 包含页面切换和平滑过渡动画
- **用户友好**: 清晰的导航和直观的操作流程

## 依赖

- PyQt6 >= 6.0
- Python >= 3.8

## 使用方法

```bash
python -m ui.gui
```
